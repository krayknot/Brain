  15.1 Drop-Down Navigation Menu  
  Eg.1 You can use the following example to create a drop-down navigation menu. These are often useful when you need to preserve space on your web pages. Place the following code in red between the head tags of your page (if you are embedding it in your web pages) or place it in your external script document (in which case you should leave out the opening and closing script tags). Then reference the external script document in your web pages.
 
<script language="javascript">
<!-- Hide from older browsers
function load() {       
 var URL = document.navmenu.site.selectedIndex
 location.href = document.navmenu.site.options[URL].value
 }
 // End Hide -->
</script>

Now place the following code in red between the body tags of your document at the place where you want the drop-down navigation menu to appear. This drop-down menu does not have a Go button, i.e. once a selection is made, it is immediately followed and the new web page starts to load. The following two examples contain relative URI's which are links to web pages on the same site. If your links are to external sites then you must use absolute URI's, e.g. <option value="http://www.cnn.com">CNN for the option values.

 
<form name="navmenu">
<select name="site" size="1" onChange="load(this)" style="color: #ffffff; 
font-family: verdana; font-weight: bold; font-size: 10px; background-color: #777777">
<option value="">Navigation Menu
<option value="index.htm">Homepage
<option value="products.htm">Our Products
<option value="prices.htm">Our Prices
<option value="specials.htm">Special Offers
<option value="newsletter.htm">Free Newsletter
</select>
</form>

OR

Use this variation of the above code between the body tags if you want a drop-down navigation menu with a Go button. With this type of navigation menu a user has to click on the Go button after making a selection before the new web page starts to load.

 
<form name="navmenu">
<select name="site" size="1" style="color: #ffffff; 
font-family: verdana; font-weight: bold; font-size: 10px; background-color: #777777">
<option value="">Navigation Menu
<option value="index.htm">Homepage
<option value="products.htm">Our Products
<option value="prices.htm">Our Prices
<option value="specials.htm">Special Offers
<option value="newsletter.htm">Free Newsletter
</select>
<input type="button" value="Go" onClick="load(this)">
</form>
  
     
  15.2 Button To Close A Pop-Up Window  
  Eg.2 Use the following example to create a button that will close a pop-up or secondary browser window. Place the script between the body tags where you want the button to appear on the page.
 
<center>
<form name="closebutton">
<input type="button" value="CLOSE" onClick="window.close()">
</form>

</center>

  
     
  15.3 Text Link To Close A Pop-Up Window  
  Eg.3 Use the next example to create a text link, that when clicked, will close the pop-up or secondary window. Again, place the code between the body tags where you want the text link to appear on the page.
<center><a href="javascript:window.close()">CLOSE WINDOW</a></center>

In both Eg.2 and Eg.3 above, the center element has been used to horizontally center the button and text link on the page.

 

  
     
  15.4 Adding A Back Button  
  Eg.4 If you've got a web page on your site that is accessed from many different web pages, such as a copyright notice and you don't want to add a link back to all the other pages that link to it, then just add a javascript back button that links to the previous page using the following code which is placed between the body tags:
 
<center>
<form name="backbutton">
<input type="button" value="BACK" onClick="history.back()">
</form>
</center>

Eg.5 Or you can customize the look of the button by adding inline style information as follows (works with any form button):

 
<center>
<form name="backbutton">
<input type="button" value="BACK" onClick="history.back()" style="font-family:verdana; 
font-size:12px; font-weight:bold; color:#000000; background-color:#00FFFF">
</form>

</center>

  
     
  15.5 Disabling Browser Caching  
  Eg.6 If you don't want browsers to cache a particular web page or web pages, perhaps because they may be time sensitive (stocks and shares information for example), you can add the following meta tag between the head tags of the document(s) you wish to prevent from being cached as follows:
<head>
<meta http-equiv="pragma" content="no-cache">
</head>

This will prevent your web page from being cached by most browsers, however it will have no effect on proxy server caches.

Whatever your reason is for preventing caching, make sure it's a very good one. Remember that a lot of people go back and refer to web pages of interest to them after a surfing session and that most of these people are probably using a dial-up connection. If your web page prevents caching it will not be available offline and will have to be downloaded again. Therefore, disabling browser caching without a good reason could be very annoying to most people.

  
     
  15.6 Browser Redirect Page  
  Eg.7 Use this code if you no longer have a particular web page on your site and want to avoid your visitors from getting 404 error messages. You can easily redirect the browser to another web page on your site or even to another site. Simply replace the old page with this HTML code:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">
<html>
<head>
<meta http-equiv="refresh" content="1;URL=name_of_page.htm">
</head>
<body>
<p><center>
If you are not redirected automatically, click <a href="name_of_page.htm">here</a>.
</center></p>
</body>
</html>

If the redirection is to another page on your web site then the value of URL would be URL=name_of_page.htm (if the page extension that you are using on your site is different, replace .htm with the extension that you use on your site, i.e. .html, .sht or .shtml). However, if the redirection is to another site, then include the full URL to the external site as follows: URL=http://www.sitename.com/. The 1 in the meta tag instructs the browser to wait 1 second before redirection commences. You can change this value to whatever suits you.

You should also add a link to the destination web page or external site as well, since some older browsers don't support the refresh meta tag. (The code for the link is shown in blue between the body tags in the above example; it can be written on one line but not done so here to prevent this page from having to be scrolled horizontally by the reader. Be sure to include the correct URL here also).

 

  
     
  15.7 Break Out Of Frames  
  Eg.8 Using this script on your page(s) will prevent other sites (i.e. framed sites) from displaying your page(s) within their frame. However, do not use this script on pages that are displayed within your own frames otherwise the script will destroy the frames. To automatically break your page(s) out of frames, add the following onload event handler inside the opening body tag of your web page(s) as follows:
<body onLoad="if(self!=top)top.location=self.location">
...Your page contents here...

</body>

  
     
  15.8 Automatically Add Page Modification Date  
  Eg.9 Use this script to automatically add a page last modified date and time on a web page. The date and time information is retrieved from the file's date and time stamp since the system automatically changes the date and time stamp on the file whenever a file is modified and saved. Place the following script between the body tags exactly where you want the page modification details to appear on the web page:
 
<script language="javascript" type="text/javascript">
<!--
document.write("This page was last modified on " + document.lastModified)
// -->
</script>

Make sure you include the single blank space between on and " (which is part of the sentence between the brackets above).

 

  
     
  15.9 Creating A Popup Window  
  There are three variations of this script.
  1. One to create a popup window on your visitor's browser when a link or button is clicked,
  2. One to create a popup window when a web page loads and
  3. One to create a popup window when a web page unloads.

Eg.10 Create a popup window when a link or button is clicked.

Place the first part shown below between the head tags of the document:

<script language="javascript">
function popupwin(){
window.open('URL','WindowName','toolbar=1,menubar=1,scrollbars=1,resizable=1,status=1,
          location=1,screenX=120,screenY=160,top=120,left=160,width=470,height=350')
}
</script>

Note: The line in blue above must be typed on one line or the script will not work. It's been written on two lines here to prevent readers from having to scroll this page horizontally.

You can edit what's in between the brackets after window.open as follows (but don't remove the single quotation marks):

  1. Replace URL with the location of the web page that you want to load in the popup window. If the web page is in the same directory you can use a relative URL such as page_name.htm
    If it's located at an external site then you must use the full URL such as     http://www.othersite.com (this will load the site's index [default] page) or http://www.othersite.com/page_name.htm (this will load the page that you have specified).
  2. Replace WindowName with whatever name you choose to give the popup window. If it's to prompt your visitor's to sign up for your newsletter then you can name the popup window 'newletter'. This is usually only used in situations where you would want to target other links to load in that particular popup window. Otherwise you can leave the space between the single quotation marks blank but with no white space between them as follows '' However, do not remove the quotation marks from the line of code.
  3. The popup window properties of toolbar through to location have been given a value of 1 which is equivalent to yes, i.e. the popup window will have all of these properties. Therefore a popup window created using the above script will have a toolbar, a location bar and all the other properties listed in between these two properties.

    Giving these properties a value of     0 (zero) is equivalent to no. The popup window will not have any properties that are given a value of 0.

    You can therefore customize the properties of the popup window by giving each property a value of 1 or 0.

    For     scrollbars, you can also use a value of auto, in which case the popup window will only have scrollbars if they are needed, i.e. if the web page being displayed is larger than the viewing area of the popup window and therefore needs to be scrolled.
  4. screenX, screenY, top and left refer to the screen position of the popup window when it opens. ScreenX and screenY is supported by Netscape and top and left is supported by Internet Explorer. Change their values to suit your own situation.

    Remember that     screenX is equivalent to top (both of which refer to the screen's x-axis and must have the same value) and that screenY is equivalent to left (both of which refer to the screen's Y-axis and must also have the same value).
  5. The width refers to the width of the popup window. You can change this value to suit your own needs.
  6. The height refers to the height of the popup window. You can change this value to suit your own needs.

To create a clickable link, place the second part shown below between the body tags of your document and position it where you want it to appear on the web page:

<a href="javascript:popupwin()">link description</a>

OR

To create a clickable button, place the second part shown below between the body tags of your document and position it where you want it to appear on the web page:

<form name="popupwindow">
<input type="button" value="LABEL HERE" onclick="popupwin()">
</form>

Change LABEL HERE to whatever text you want to appear on the button.

  15.10 Automatically Close A Popup Window  
  Eg.13 Use the following script to automatically close a popup window after a certain number of seconds. Add the following onload event handler inside the opening body tag of the web page that is going to load in the popup window as follows:
<body onload="setTimeout(window.close, 10000)">
...Popup page's contents here...
</body>

The numerical value of 10000 refers to the number of milliseconds that the popup window should stay open for before closing. Change this numerical value to suit your own situation. Remember that this figure is always given in milliseconds.

1000 milliseconds = 1 second
10000 milliseconds = 10 seconds
