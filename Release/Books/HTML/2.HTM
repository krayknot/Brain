  3.1 Components of an HTML Document  
 

3.11 Elements

HTML elements and their tags are used to construct and format HTML pages. For example, the i element represents italic text while the p element represent a paragraph. HTML includes element types that represent paragraphs, hypertext links, lists, tables, images, and all the other constructs that make up a web page.

An element has three parts as follows: a start tag, an end tag and the content that goes between the start and end tags. Tags are delimited by "<" and ">". An end tag must include a forward slash, i.e. "/", after the "<". For example, the i element has a start tag, <i>, and an end tag, </i>. The start and end tags surround the content of the i element, for example:

<i>This is how to get italic text</i>

In HTML, element names are case-insensitive, so <i>, and <I> have the same effect.

In HTML, some elements allow the end tag to be omitted (e.g., the p and li element types). The li end tag, for example, is always optional when coding in HTML since the element's end is implied by the next li element or by the end of the list:

<ul>
<li>List item one; no end tag
<li>List item two; optional end tag included</li>
<li>List item three; no end tag
</ul>

Some elements have no end tag because they have no content. These elements, such as the br element, whose only role is to terminate a line of text, are represented only by a start tag. These elements are said to be empty and never have end tags.

Elements are not tags. Some people refer to elements as tags (e.g., "the i tag"). Remember that i is an element, and the tag, be it the start tag (<i>), or the end tag (</i>), is another.

3.12 Attributes

Attributes are used to define the properties of elements. Each element has its own set of attributes. For example, src is an attribute of the <img> tag, which indicates the location and file name of an image file and the alt attribute which gives alternate text for those whose browsers are configured not to load images, for example:

<img src="backbutton.gif" alt="Click to go to previous page">

An attribute is included in the start tag only and not in the end tag. It takes the form
 attribute-name="attribute-value". The attribute value is delimited by single or double quotation marks. Single quote marks can be included within the attribute value when the value is delimited by double quote marks, and vice versa. In HTML the quotation marks are optional if the attribute value consists solely of letters (a-z or A-Z), digits (0-9), hyphens ("-"), periods ("."), underscores ("_") and colons (":"). In HTML attribute names are always case insensitive.

3.13 Comments

<!-- -->

In HTML, comments are ignored by the HTML parser. All text between the beginning of the comment (<!--) and the end of the comment (-->) is ignored. White space is allowed within the comment but not between the opening <!-- and closing -->. Both the start and end tags are required. A common error is to include a string of hyphens ("---") within a comment. You should avoid putting two or more adjacent hyphens inside comments.

<!-- This is how you write a comment in HTML, by placing it between the comment tags -->

3.14 Special Characters

As an alternative to entities, you can also use numeric character references. Any character may be represented by a numeric character reference based on its "code" in Unicode.   3.2 Example HTML Formatted Document  
  An HTML document begins with a DOCTYPE declaration that declares the version of HTML to which the document corresponds to. Different versions may be used depending on the level of compatibility the author wishes to indicate. Following that is the HTML element which contains the HEAD and BODY elements. The HEAD contains information about the document, such as its title and meta-tags (giving information about keywords and description), while the BODY contains the actual content of the document. A basic HTML document takes on the following format:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
  <head>
    <title>Document title goes here</title>
  </head>
  <body>
  <h1>Page heading</h1>
  <p>...First paragraph here...</p>
  <p align="center">...Second paragraph here...</p>
  <ul>
    <li>List item one.</li>
    <li>List item two.</li>
  </ul>
  <img src="image_name.gif" width="40" height="40" alt="This is an image" border="0">
  </body>
</html>

In a Frameset document, the frameset element replaces the body element. (Discussed in the section on Frames).
